# Thay đổi v1.3.0 — Một sheet / Nhiều sheet

- Card Google Sheet đích nằm trên cùng.
- Bỏ phần mô tả tên file, cách nhập và các checkbox tùy chọn.
- Dòng đầu CSV luôn được xem là tiêu đề.
- Luôn thay thế toàn bộ dữ liệu tab đích.
- **Nhiều sheet:** suy ra tab từ `[Tên Google Sheet] - [Tên tab].csv`.
- **Một sheet:** nhập tên tab trực tiếp và chỉ dùng file đầu tiên trong danh sách.
- Bảng CSV hiển thị rõ tab đích hoặc trạng thái `Không nhập`.
